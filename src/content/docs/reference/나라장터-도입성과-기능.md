# 나라장터(KONEPS) — 도입 성과·7가지 유형·5대 기능

## 개요

나라장터(KONEPS; Korea ON-line E-Procurement System)는 중앙행정기관 및 공공기관이 사용하는 **국가종합전자조달시스템**으로, 공공조달의 단일 창구(Single Window) 역할을 담당한다. [[공공조달-기본원칙-구성요소|투명성·공정성·경쟁촉진 원칙]]을 제도 인프라로 구현한 핵심 시스템이다.

> [!note] 도입 배경 — 2002년 이전 무엇이 문제였는가?
> 나라장터 이전 공공조달은 **서류 중심·수작업** 방식이었다. 조달기업은 계약 서류 제출을 위해 공공기관을 직접 방문해야 했고, 각 기관마다 별도 등록 절차가 있었다. 더 심각한 문제는 **조합이 소속 회원사에게 물량을 임의 배정**하는 담합 구조였다 — 기술력이 아니라 조합 내 정치력과 유력 업체와의 연결이 수주를 결정했다. 나라장터는 입찰 공고부터 계약체결까지 전 과정을 온라인에서 실시간 공개함으로써 이 구조를 해체했다. 조달기업은 1회 등록으로 모든 공공기관 입찰에 참가할 수 있게 됐다.

## 현행 규정

### 나라장터 도입 성과 3가지

| 성과 | 내용 |
|------|------|
| **거래비용 절감** | 입찰·계약 서류 제출 위해 공공기관 방문 불필요; 여비 등 **8조 원** 상당 거래비용 절감 |
| **투명한 조달행정** | 조달정보 실시간 공개, 표준절차 업무처리 → 국제투명성기구·세계 반부패포럼 등에서 **투명성 우수사례** 소개 |
| **중소기업 경쟁력 제고** | 공공입찰 정보 통합 공고로 참여기회 확대·마케팅 비용 절감; 나라장터 등록업체(50만여 개) 중 **중소기업 96.7%** |

> [!note] 3가지 성과와 [[공공조달-기본원칙-구성요소|기본원칙]] 연결
> - **거래비용 절감** → [[공공조달-기본원칙-구성요소|경제성 원칙]] 구현: 방문·서류 비용 제거
> - **투명한 조달행정** → [[공공조달-기본원칙-구성요소|투명성·공정성 원칙]] 구현: 실시간 정보 공개, 표준 절차
> - **중소기업 경쟁력 제고** → [[공공조달-기본원칙-구성요소|경쟁촉진 원칙]] 구현: 담합·조합 배분 구조 해체, 전국 단위 입찰 참가 가능

> [!example] 투명성 성과 — OECD 반부패포럼 우수사례 (2004)
> 나라장터는 2004년 OECD 반부패포럼에서 **전자조달 투명성 우수사례**로 선정됐다. 조달청의 청렴도 측정 결과는 2002년 6.8점에서 2005년 8.65점으로 27% 상승했으며, **2004년 이후 현재까지 업무 관련 비위 징계가 단 한 건도 발생하지 않았다.** 입찰 정보가 실시간 공개되고 표준 절차가 강제되면서 조작 여지 자체가 사라진 결과다. 동남아시아·중앙아시아·아프리카 등 50여 개국에 구축 노하우를 전수하는 수출 모델이 됐다.

### 나라장터 연계 7가지 플랫폼

1. 나라장터
2. 혁신장터
3. 종합쇼핑몰
4. 벤처나라
5. 디지털서비스몰
6. 이음장터
7. 해외조달장터

### 전자조달시스템 5대 기능

| 기능 | 내용 |
|------|------|
| **전자입찰** | 입찰 공고 → 입찰참가 신청 → 투찰 → 개찰 → 낙찰자 선정 |
| **전자계약** | 공동인증·생체지문인식 기술 적용; 계약서 작성·전자서명·보증서·인지세·채권 등 처리 |
| **전자지불** | 검사·대금청구·지불까지 국가·지자체·교육기관 3대 재정정보시스템 연계 처리 |
| **종합쇼핑몰** | 조달기업 상품 등록; 공공기관이 직접 검색·구매 |
| **목록정보** | 406만 개 품목 정보를 UNSPSC 분류체계 활용; 물품·서비스 규격·품질 정보 공유 |

> [!note] 왜 전자협상은 나라장터 기능이 아닌가?
> 협상은 양 당사자 간 실시간 대화·제안·반론이 오가는 과정이다. 나라장터의 5대 기능(입찰·계약·지불·쇼핑몰·목록정보)은 모두 **표준화된 절차를 자동화**하는 것이지만, 협상은 사안마다 내용이 달라 표준 절차로 담기 어렵다. 전자협상은 별도 시스템 또는 오프라인으로 진행된다.

## 시험 출제 포인트

**출제 패턴:** "나라장터(국가종합전자조달시스템) 도입 성과"
- 3가지: 거래비용 절감(8조 원), 투명한 조달행정(국제 우수사례), 중소기업 경쟁력 제고(96.7%)

**출제 패턴:** "나라장터의 기능으로 옳지 않은 것"
- **전자협상** → **오답** (나라장터 기능 아님)
- 옳은 것: 전자입찰, 전자계약, 전자지불

**핵심 암기:**
- 나라장터 = 7만 1천여 공공기관 + 60만여 조달업체 이용
- 도입 성과 수치: 8조 원 거래비용 절감, 중소기업 96.7%
- 7가지 플랫폼 목록 암기
