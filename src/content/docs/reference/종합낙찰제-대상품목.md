# 종합낙찰제 — 대상품목 3개 그룹 및 낙찰자 결정

## 개요

물품의 제조·구매 계약에서 입찰가격 외에 품질·에너지효율·환경성 등을 종합적으로 참작하여 낙찰자를 결정하는 제도이다. **추정가격 5천만 원 이상**인 특정 품목에 적용된다.

> [!note] 왜 이 제도가 존재하나?
> 일반 적격심사제나 최저가낙찰제에서는 제품의 에너지소모비와 환경성능이 낙찰 평가에 반영되지 않는다. 종합낙찰제는 **제품 품질향상 및 에너지절약·환경우수 제품 개발을 유도**하기 위해 도입되었다. 에너지 소비효율 평가제품의 경우 구매 가격뿐 아니라 운용 기간 중 발생하는 **총에너지 소모비를 합산**하여 실질적인 총비용을 기준으로 낙찰자를 선정한다. 이를 통해 초기 가격이 약간 높더라도 운용비용이 낮은 고효율 제품이 낙찰될 수 있다.

> [!info] 종합낙찰제 vs 종합심사낙찰제 — 이름이 유사하나 완전히 다른 제도
> - **종합낙찰제** → [[종합낙찰제-대상품목|물품]] 전용 (에너지·환경 15종, 5천만 원 이상)
> - **[[종합심사낙찰제-대상|종합심사낙찰제]]** → 공사·건설기술용역 전용 (100억 이상 공사 등)
> 시험에서 두 제도를 혼용하는 오답 보기가 자주 출제된다.

## 현행 규정

### 대상 품목 3개 그룹

#### ① 에너지 소비효율 평가제품 (6종)

추정가격 5천만 원 이상

| 대상 품목 |
|---|
| 펌프 |
| 냉동기 |
| 공기압축기 |
| 송풍기 |
| 엘리베이터 |
| 에스컬레이터 |

**낙찰자 결정:** 총에너지 소모비 심사 후 → (에너지 소모비 + 입찰금액) 합산 최저가 업체 선정

> [!example] 에너지 소모비 합산 낙찰 사례
> A사 입찰가 4,800만 원 / 총에너지 소모비 600만 원 → 합산 5,400만 원
> B사 입찰가 4,500만 원 / 총에너지 소모비 1,200만 원 → 합산 5,700만 원
> → **A사 낙찰** (입찰가는 낮아도 에너지 비용 포함 시 B사가 더 비쌈)

#### ② 환경우월성기준 적용제품 (4종)

추정가격 5천만 원 이상

| 대상 품목 |
|---|
| 에어컨 |
| 세탁기 |
| LCD모니터 |
| 데스크탑컴퓨터 |

#### ③ 환경우월성기준 확대적용제품 (5종)

추정가격 5천만 원 이상

| 대상 품목 |
|---|
| 노트북 |
| 텔레비전 |
| 프린터 |
| LED램프 |
| 공기청정기 |

**② + ③ 낙찰자 결정:** 업체별 가격·성능·환경환산점수로 계산한 **종합평가점수 최고점수** 업체 선정

### 낙찰자 결정 기간

- 낙찰자 결정은 특별한 사유가 없는 한 **개찰일로부터 10일 이내**

### 낙찰자 결정 방식 요약

| 그룹 | 낙찰 기준 |
|---|---|
| 에너지 소비효율 평가제품 (6종) | (에너지 소모비 + 입찰금액) 합산 **최저가** |
| 환경우월성기준 적용제품 (4종) | 가격·성능·환경환산점수로 산출한 종합평가점수 **최고점** |
| 환경우월성기준 확대적용제품 (5종) | 가격·성능·환경환산점수로 산출한 종합평가점수 **최고점** |

## 적용 조건

- 물품 제조·구매 계약에만 적용 (용역·공사는 [[종합심사낙찰제-대상|종합심사낙찰제]] 적용)
- 추정가격 5천만 원 이상인 대상 품목에 한정

> [!warning] "낙찰자 결정 기준이 세 그룹 모두 동일하다" — 오답
> 에너지 소비효율 6종은 **합산 최저가** 방식, 환경우월성 9종은 **종합평가점수 최고점** 방식으로 낙찰 기준이 다르다.

## 시험 출제 포인트

**출제 패턴:** 에너지 소비효율 평가제품 대상 품목 6종 암기 — "이 품목이 포함되는가?" 또는 "몇 종인가?" 형태로 출제.

**핵심 암기 — 그룹 구분:**
- **에너지효율 6종:** 펌프·냉동기·공기압축기·송풍기·**엘리베이터**·에스컬레이터
- **환경우월성 4종:** 에어컨·세탁기·LCD모니터·데스크탑컴퓨터
- **확대적용 5종:** 노트북·텔레비전·프린터·LED램프·공기청정기

**오답 유인 (Q8 기출 패턴):**
- 엘리베이터를 환경우월성 제품으로 혼동 — **엘리베이터는 에너지효율 6종**
- 에어컨을 에너지효율 제품으로 혼동 — 에어컨은 환경우월성 4종
- 데스크탑컴퓨터와 노트북을 같은 그룹으로 혼동 — 데스크탑=4종, 노트북=5종(확대)
- 종합낙찰제를 공사·용역에도 적용한다고 혼동 — 물품만 적용
