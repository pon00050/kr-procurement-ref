# 공공조달 규정 레퍼런스

국가계약법·지방계약법·조달청 훈령·계약예규를 법원(法源) 원문에서 직접 인용한 규정 카드 모음입니다.
교재 요약이 아닌, 법제처(law.go.kr) 원문과 조달청 고시를 직접 인용하며 근거 조문을 명시합니다.

---

## 탐색 방법

왼쪽 사이드바에서 세 가지 방법으로 탐색할 수 있습니다.

| 방법 | 사이드바 섹션 | 적합한 상황 |
|------|-------------|------------|
| **워크플로우별** | 워크플로우별 | 업무 단계(입찰·낙찰·계약이행 등)에서 관련 규정을 찾을 때 |
| **계약 유형별** | 계약 유형별 | 물품·용역·공사·MAS 계약에 해당하는 규정을 찾을 때 |
| **시험 과목별** | 시험 과목별 | 공공조달관리사 과목1·2·3 특정 장(章)을 공부할 때 |

검색창(Ctrl+K)에서 용어를 직접 입력해도 됩니다 — 예: "낙찰하한율", "적격심사", "동시이행".
