## 이 레퍼런스는 무엇인가요?

국가계약법·지방계약법·조달청 훈령·계약예규를 바탕으로 작성한 **공개 규정 카드 모음**입니다.
교재 요약이나 수험 강의록이 아닌, 법제처 원문(law.go.kr)과 조달청 고시를 직접 인용합니다.
각 카드는 근거 조문(`governing_articles`)과 검증 날짜(`last_verified`)를 명시합니다.

## 세 가지 탐색 방법

| 방법 | 적합한 상황 | 경로 |
|------|------------|------|
| **시험 과목별** | 과목1·2·3 특정 장(章)을 공부할 때 | 왼쪽 사이드바 → 시험 과목별 |
| **워크플로우별** | 입찰·낙찰 등 업무 단계에서 규정을 찾을 때 | 왼쪽 사이드바 → 워크플로우별 |
| **검색** | 용어를 알고 있을 때 — 예: "낙찰하한율", "적격심사" | 상단 검색창 (Ctrl+K) |

## 업데이트 방침

- 모든 카드는 `last_verified` 날짜와 근거 조문을 표시합니다.
- 법령 개정 감지 시 30일 이내 반영합니다 (법제처 API 자동 폴링).
- 개정 이력은 [변경 로그](/changelog/)에서 확인하세요.
- 오류나 누락을 발견하셨다면 → [이슈 제보](#) (링크 추가 예정)

## 실무에서 이 레퍼런스를 활용하시나요?

규정을 아는 것과 **클라이언트별로 적용하는 것**은 다릅니다.
[공공조달관리사 워크플로우 플랫폼](https://kr-public-procurement-demo.up.railway.app)에서는:

- 적격심사 점수 자동 계산 (클라이언트별)
- 규제 변경 알림 (관련 계약에 영향을 미치는 개정만 필터링)
- 계약 이행 이력 관리 및 감사 기록

레퍼런스 카드가 "무엇인가"를 알려준다면, 플랫폼은 "어떻게 적용하는가"를 자동화합니다.
